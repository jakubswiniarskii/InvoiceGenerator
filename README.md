Invoice PDF Generator

This application is designed to generate a large number of invoice PDFs automatically from data stored in an Excel file.
It helps automate repetitive invoicing tasks and ensures consistent formatting and naming of generated documents.

Features:
- Import invoice data from Excel
- Bulk generation of PDF invoices
- Automatic output folder creation
- Customizable PDF file naming
- Data validation (missing fields, invalid formats)
- Generation logs and error reporting
- Consistent invoice layout using templates
